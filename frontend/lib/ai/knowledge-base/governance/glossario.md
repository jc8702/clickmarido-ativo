# Glossário de Termos

## Termos do Sistema

| Termo | Definição |
|-------|-----------|
| **CRM** | Customer Relationship Management - Sistema de gestão de relacionamento com clientes |
| **OS** | Ordem de Serviço - Documento que acompanha a execução de um serviço |
| **SKU** | Stock Keeping Unit - Código único para identificar produtos/serviços |
| **Dashboard** | Painel de controle com indicadores e gráficos |
| **Backend** | Parte do sistema que roda no servidor |
| **Frontend** | Parte do sistema que o usuário vê e interage |
| **API** | Application Programming Interface - Interface de programação |
| **JWT** | JSON Web Token - Padrão de autenticação |
| **LGPD** | Lei Geral de Proteção de Dados |

## Termos de Serviços

### Elétrica
| Termo | Definição |
|-------|-----------|
| **Disjuntor** | Dispositivo de proteção que interrompe o circuito em caso de sobrecarga |
| **Quadro de força** | Painel que concentra os disjuntores da instalação |
| **Fiação** | Conjunto de fios elétricos da instalação |
| **Aterramento** | Conexão com a terra para segurança |
| **Curto-circuito** | Conexão acidental entre fios de mesma polaridade |

### Hidráulica
| Termo | Definição |
|-------|-----------|
| **Registro** | Válvula que controla o fluxo de água |
| **Vazamento** | Perda de água por falha na tubulação |
| **Goteira** | Vazamento pequeno e contínuo |
| **Desentupimento** | Remoção de obstruções em encanamentos |
| **Caixa d'água** | Reservatório de água |

### Automação
| Termo | Definição |
|-------|-----------|
| **Smart Home** | Casa inteligente com dispositivos conectados |
| **Domótica** | Automação residencial |
| **Zigbee** | Protocolo de comunicação sem fio para dispositivos |
| **Z-Wave** | Protocolo de comunicação sem fio alternativo |
| **Hub** | Central de controle da automação |

### Móveis
| Termo | Definição |
|-------|-----------|
| **Montagem** | Processo de assembler um móvel |
| **Desmontagem** | Processo de desaser um móvel |
| **Dobradiça** | Peça que permite abertura de portas/gavetas |
| **Puxador** | Peça para abrir portas/gavetas |
| **Ferragem** | Conjunto de peças metálicas do móvel |

## Termos Técnicos

| Termo | Definição |
|-------|-----------|
| **SLA** | Service Level Agreement - Acordo de nível de serviço |
| **Fallback** | Opção alternativa quando a principal falha |
| **RAG** | Retrieval-Augmented Generation - Geração aumentada por recuperação |
| **KB** | Knowledge Base - Base de conhecimento |
| **Intent** | Intenção detectada na mensagem do usuário |
| **Escalonamento** | Transferência para atendimento humano |
| **Provider** | Provedor de serviços (ex: OpenRouter, Kilo) |
| **Rate Limit** | Limite de requisições por período |

## Termos de Pagamento

| Termo | Definição |
|-------|-----------|
| **PIX** | Sistema de pagamento instantâneo do Banco Central |
| **Boleto** | Documento de cobrança bancária |
| **Mercado Pago** | Gateway de pagamento integrado |
| **Fatura** | Documento fiscal de cobrança |
| **NF** | Nota Fiscal |

## Termos de Negócio

| Termo | Definição |
|-------|-----------|
| **Orçamento** | Proposta de serviços com valores |
| **Aceito** | Orçamento aprovado pelo cliente |
| **Rejeitado** | Orçamento recusado pelo cliente |
| **Expirado** | Orçamento não respondido em 7 dias |
| **Garantia** | Período de cobertura após serviço |
| **SLA** | Acordo de prazo e qualidade |
