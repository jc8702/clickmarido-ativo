# Uso Geral do Sistema

## O que é o ClickMarido?
O ClickMarido é um sistema de gestão (CRM) completo para empresas de serviços residenciais. Ele centraliza cadastro de clientes, orçamentos, ordens de serviço, pagamentos e muito mais.

## Como Acessar
1. Acesse o endereço do sistema no navegador
2. Insira seu email e senha
3. Clique em "Entrar"

## Navegação Básica
- **Sidebar lateral**: Acesso rápido a todas as seções
- **Dashboard**: Visão geral do negócio
- **Notificações**: Ícone sino no canto superior direito
- **Perfil**: Acesse configurações da conta

## Principais Funcionalidades

### Dashboard
- Visão geral de receitas e despesas
- Gráficos de tendência
- Indicadores de performance
- Acesso rápido a ações importantes

### Clientes
- Cadastro completo com dados de contato
- Histórico de serviços por cliente
- Busca por nome, email ou telefone
- Múltiplos endereços por cliente

### Orçamentos
- Criação com itens vinculados
- Cálculo automático de totais
- Envio por email/WhatsApp
- Aprovação/rejeição online

### Ordens de Serviço (OS)
- Acompanhamento do fluxo de execução
- Registro de fotos
- Agendamento com data e hora
- Conclusão automática gera pagamento

### Pagamentos
- PIX, boleto, cartão, dinheiro
- Integração com Mercado Pago
- Lembretes automáticos
- Controle de recebimentos

## Dicas de Uso
- Use atalhos de teclado (Ctrl+K para busca)
- Favorite as páginas que mais acessa
- Configure notificações por email
- Mantenha dados atualizados
