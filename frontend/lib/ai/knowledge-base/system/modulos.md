# Módulos do Sistema

## Visão Geral
O ClickMarido é dividido em módulos para organizar as funcionalidades. Cada módulo tem uma área específica de atuação.

## Módulos Disponíveis

### 1. Dashboard
- Visão geral do negócio
- Gráficos de receitas/despesas
- Indicadores de performance
- Acesso rápido a ações

### 2. Clientes
- Cadastro e edição de clientes
- Histórico de interações
- Documentos e anexos
- Busca e filtros avançados

### 3. Orçamentos
- Criação de orçamentos
- Status: Rascunho → Enviado → Aceito/Rejeitado
- Envio por email/WhatsApp
- Visualização pública (cliente)

### 4. Produtos/Serviços
- Catálogo de serviços oferecidos
- Peças e materiais
- Preços e categorias
- Controle de estoque (peças)

### 5. Ordens de Serviço (OS)
- Fluxo de execução
- Agendamento
- Registro de fotos
- Conclusão e avaliação

### 6. Pagamentos
- Gestão de cobranças
- Métodos: PIX, boleto, cartão, dinheiro
- Integração Mercado Pago
- Lembretes automáticos

### 7. Fornecedores
- Cadastro de fornecedores
- Classificação de desempenho
- Histórico de compras
- Documentação fiscal

### 8. Compras
- Pedidos de compra
- Aprovação e recebimento
- Controle de estoque
- Notas fiscais

### 9. Financeiro
- Dashboard financeiro
- Fluxo de caixa
- Despesas
- Faturas

### 10. Relatórios
- Relatórios gerenciais
- Exportação (PDF, Excel)
- Filtros por período
- Indicadores

### 11. Configurações
- Dados da empresa
- Usuários e permissões
- Integrações
- Personalização

## Como Alternar entre Módulos
1. Clique no ícone do módulo na sidebar
2. Ou use o menu superior
3. Cada módulo mantém seu próprio estado

## Permissões
- Administrador: acesso total
- Gerente: acesso à maioria dos módulos
- Atendente: módulos operacionais
- Técnico: OS e agenda
- Cliente: portal próprio (visualização)

## Dicas
- Use o dashboard para visão geral
- Favorite módulos que mais usa
- Mantenha dados atualizados em todos os módulos
