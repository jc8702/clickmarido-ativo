# Erros Comuns do Sistema

## Erros de Autenticação

### "Email ou senha inválidos"
- Verifique se o email está correto
- Verifique se a senha está correta (maiusúsculas/minúsculas)
- Use "Esqueci minha senha" se necessário
- Limpe o cache do navegador

### "Sessão expirada"
- Faça login novamente
- Sessões expiram após 7 dias
- Marque "Lembrar de mim" para persistir

### "Acesso negado"
- Verifique suas permissões
- Entre em contato com o administrador

## Erros de Navegação

### "Página não encontrada"
- Verifique a URL no navegador
- Volte ao dashboard
- Use o menu de navegação

### "Erro ao carregar página"
- Verifique sua conexão com a internet
- Recarregue a página (F5)
- Limpe o cache do navegador

## Erros em Formulários

### "Dados obrigatórios não preenchidos"
- Preencha todos os campos marcados com *
- Verifique formatos (email, telefone, CPF/CNPJ)

### "Email já cadastrado"
- Use outro email
- Ou busque o cliente existente

### "Valor inválido"
- Use apenas números para valores
- Separe decimais com vírgula (ex: 1.500,00)

## Erros de Conexão

### "Erro ao conectar com o servidor"
- Verifique sua conexão com a internet
- Aguarde alguns minutos e tente novamente
- Se persistir, abra um chamado técnico

### "Tempo de conexão esgotado"
- A internet pode estar lenta
- Tente novamente em alguns minutos
- Verifique se outros sites funcionam

## Erros em Orçamentos

### "Não foi possível criar orçamento"
- Verifique se o cliente está cadastrado
- Adicione pelo menos um item
- Verifique se os valores estão corretos

### "Status inválido"
- Verifique o fluxo: Rascunho → Enviado → Aceito/Rejeitado
- Não é possível voltar status

## Erros em OS

### "OS não encontrada"
- Verifique o número da OS
- Use busca por cliente ou período

### "Não foi possível concluir OS"
- Verifique se todas as fotos foram registradas
- Preencha campos obrigatórios

## Erros em Pagamentos

### "Erro ao gerar pagamento"
- Verifique se há dados faltando
- Confirme o método de pagamento
- Se PIX, verifique a chave

### "Boleto não gerado"
- Verifique dados do cliente
- Confirme o valor
- Tente novamente

## Soluções Gerais

### Limpar Cache do Navegador
1. Pressione Ctrl+Shift+Delete
2. Selecione "Imagens e ficheiros em cache"
3. Clique em "Limpar"

### Atualizar Página
- Windows/Linux: F5 ou Ctrl+R
- Mac: Cmd+R

### Testar Conexão
- Acesse outro site
- Verifique se o Wi-Fi está funcionando

### Reportar Bug
1. Tire um print da tela
2. Anote os passos que deu
3. Abra um chamado técnico
4. Descreva o problema detalhadamente
