## 🎯 PALETA DE CORES (Visão Geral)

| Cor | Primária | Uso |
|-----|----------|-----|
| 🟣 **ROXO** | #5D3FD3 | Branding, Headers, CTAs principais |
| 🟢 **VERDE** | #10B981 | Sucesso, Aprovações, Confirmações |
| 🟠 **LARANJA** | #F59E0B | Alertas, Warnings, Atenção |

### Gradientes Assinatura
- **Hero** (Roxo → Verde): Headers, seções principais
- **Subtle** (Roxo Claro → Laranja Claro): Cards, backgrounds
- **Accent** (Verde → Laranja): Botões, elementos destacados
- **Dark** (Roxo Escuro → Verde Escuro): Navigation, footers
- **Warning** (Laranja → Vermelho): Erros, status críticos

## 🎨 ANIMAÇÕES E TRANSIÇÕES

```
fade-in:         0.3s - Entrada suave
slide-down:      0.3s - Descida do topo
slide-up:        0.3s - Subida de baixo
scale-in:        0.2s - Aparição com zoom
bounce-subtle:   2s - Flutuação sutil (loop)

Transitions:
  fast:    150ms - Hover feedback rápido
  base:    300ms - Padrão (RECOMENDADO)
  slow:    500ms - Mudanças importantes
```
